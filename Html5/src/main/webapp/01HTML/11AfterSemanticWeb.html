WEB

- 인터넷 통신망 위에서 작동하는 서비스

- 1991 년 팀 버니스리는 인터넷 공간 안에서 문서가 서로 이동할 수 있는 하이퍼링크라는
특수한 기능을 사용
문서와 문서 사이를 이동할수 있는 개념을 제안, 이를 바탕으로 World Wide Web 을 개발 배포

- 1994년 팀 버니스리는 HTML 웹 표준을 지정하는 W3C(World Wide Web Consortium)재단
창설 HTML(1991) ~ HTML5(2014) 발표

- W3C는 HTML 표준만 제정 강제하지 않으며, 이로인한 웹브라우저 전쟁(?) 이 시작 됨

HTML : Hypertext Markup Language

- 하이퍼텍스트는 사용자의 선택에 따라 특정한 정보화 관련된 부분으로 이동할 수 있는 조직화된 문서

- 문자를 동적으로 연결 , 사용자의 선택에 따라 특정한 정보, 관련된 부분으로 이동할 수 있게 조직화 표준화된 문서

- Hyper+Text (활동적, 이동성+문자) : 문자를 동적으로 연결 , 사용자의 선택에 따라 특정한 정보, 관련된 부분으로 이동

- Markup Language : 조직화 된 문서 , 표준화된 문서
